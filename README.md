# Qcord - Streaming & Recording Software

Мощное приложение для записи и стриминга на несколько платформ одновременно.

## Поддерживаемые платформы
- 📺 **Twitch** - прямой стрим на Twitch
- 🎥 **YouTube Live** - прямая трансляция на YouTube
- ✈️ **Telegram** - отправка видео в Telegram каналы и чаты
- 🎙️ **Локальная запись** - сохранение видео на диск в различных форматах

## Основные возможности
- ✨ Многоисточниковый захват (экран, вебкамера, аудио)
- 🔄 Одновременный стрим на несколько сервисов
- ⚙️ Гибкие настройки кодирования (битрейт, разрешение, FPS)
- 💾 Локальная запись при стриминге
- 🎛️ Интуитивный микшер с фильтрами
- 🔌 Поддержка плагинов
